# AI Lead Generation System

## 📌 Overview
This project is an AI-powered lead generation and website crawling system designed to automate the process of finding, extracting, and enriching business leads.

It combines web scraping, email extraction, and AI-based enrichment into a structured CRM-ready pipeline.

---

## 🚀 Features

- 🌐 Website Crawling
- 📧 Email Extraction
- 📄 HTML Saving
- 🧠 AI Lead Enrichment
- 📊 CSV Export
- 🗂 CRM Pipeline Ready
- 🗄 MongoDB Integration Ready
- 🤖 AI Automation Support
- 🔍 Lead Scoring System
- ✉️ Cold Email Generation

---

## 🛠 Tech Stack

- Python
- BeautifulSoup
- Pandas
- MongoDB
- Playwright / Web Scraping Tools
- AI APIs (Claude / LLM-based enrichment)
- Docker (for containerization)
